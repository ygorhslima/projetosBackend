# projetosBackend

## Exercícios onde pratico JavaScript/TypeScript e Express

este projeto tive ideia praticando exercícios de programação com a linguagem C na faculdade, porém, para não esquecer os meus aprendizados em JavaScript/TypeScript e como queria fazer projetos práticos com Express, então aqui eu peguei as ideias dos exercícios com C e usei uma linguagem moderna como o JavaScript/TypeScript e usar junto do Express para simular um servidor e o prompt-sync, sendo usado para a entrada de dados

este repositório é onde pratico lógica de programação e focado para backend

para fazer as mesmas configurações iniciais que fiz:

npm init -y
npm i express
npm i --save-dev @types/express
npm i --save-dev @types/node
npm i --save-dev ts-node
npm i --save-dev nodemon
npm i sequelize
tsc --init

pra executar com ts-node
npx ts-node teste.ts
